For Users

How to Make a Booking

  1. Login - Create an account or login to the system
  2. Choose Room - Select date, time, and room type from homepage or booking form
  3. Fill Details - Complete the booking form with your information
  4. Submit - Review and submit your booking request
  5. Confirmation - Receive booking confirmation with unique ID
  6. Wait for Approval - Admin will review and approve/reject your request

Booking Status Types

Your booking can have one of these statuses:

  • Menunggu Konfirmasi - Waiting for admin approval
  • Disetujui - Booking approved, room reserved for you
  • Ditolak - Booking rejected (room unavailable or conflict)

How to Manage Bookings

  1. Login to Admin Panel - Use credentials above
  2. View Dashboard - See all bookings and statistics
  3. Filter Bookings - Use filters (All, Pending, Approved, Rejected)
  4. Search - Find specific bookings by ID, name, email, or room
  5. Review Details - Click "View" to see complete booking information
  6. Approve/Reject - Click appropriate button for pending bookings
  7. Logout - Always logout when done

Admin Panel Features

  • Statistics Dashboard - View total, pending, approved, and rejected bookings
  • Real-time Updates - Changes reflect immediately across the system
  • Advanced Search - Find bookings quickly with search functionality
  • Detailed View - See all booking information in modal popup
  • Easy Status Management - One-click approve or reject

Important Notes

  • All data is stored in browser localStorage (for demo purposes)
  • Clear your browser cache will delete all bookings
  • Status changes are permanent and cannot be undone
  • Only pending bookings can have their status changed
  • Multiple admins can access the panel with the same credentials
